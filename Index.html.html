Installing it

  1. Tap the download button above and open the downloaded file.
  2. Android will likely block it with an "unknown source" warning — this is normal for apps outside the Play Store, not a sign anything's wrong.
  3. Tap Settings in that warning, allow installs from this source, then go back and tap the file again.
  4. Already have an older debug/test build installed? You may need to uninstall it first — a differently-signed version can't install over an existing one.
